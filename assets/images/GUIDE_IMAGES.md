# Guide rapide pour ajouter les images

Tu peux déposer toutes tes vraies images dans `assets/images/photos-a-ajouter/`, puis demander à Codex de les placer dans les bons emplacements.

## Méthode la plus simple

1. Mets toutes tes images dans `assets/images/photos-a-ajouter/`.
2. Donne des noms compréhensibles aux fichiers, par exemple :
   - `photo-profil.jpg`
   - `rafale-schema-electrique.png`
   - `rafale-routage-ares.png`
   - `rafale-vue-3d.png`
   - `rafale-percage.jpg`
   - `rafale-soudure.jpg`
   - `rafale-oscilloscope-avant.jpg`
   - `rafale-oscilloscope-apres.jpg`
3. Dans Codex, colle ce message :

```text
Toutes mes images sont dans assets/images/photos-a-ajouter/. Remplace les images provisoires du portfolio par les vraies images quand le nom du fichier correspond au contenu. Garde les images provisoires quand tu n'es pas sûr. Ne change pas la structure du site. Mets aussi à jour les textes alt des images.
```

## Correspondance conseillée

| Section | Image conseillée |
|---|---|
| Accueil | photo de profil + 3 photos de projets électroniques |
| À propos | photo naturelle ou professionnelle |
| Associations | logos ou photos d'activités |
| Centres d'intérêt | photo danse + photo guitare |
| Bénévolat | photo Hypermondes + photo Robocup |
| Projet Rafale / Présentation | carte terminée, vue 3D, schéma électrique |
| AC11.01 | cahier des charges, synoptique à 4 blocs, tableau des besoins |
| AC11.02 | calculs, composants, ISIS, ARES, routage, perçage, soudure |
| AC11.03 | dossier de fabrication, nomenclature, plan de perçage |
| AC12.01 | multimètre, oscilloscope, tableau de mesures |
| AC12.02 / AC12.03 | problème observé, oscilloscope avant/après, correction |
