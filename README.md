# Portfolio Diane Lanau

Portfolio étudiant responsive en HTML, CSS et JavaScript natif.

## Ouvrir le site

1. Dézipper le dossier.
2. Ouvrir le dossier `portfolio_diane_lanau` dans VS Code.
3. Lancer `index.html` avec l’extension Live Server, ou ouvrir directement le fichier dans un navigateur.

## Ce qui a été corrigé dans cette version

- Le menu principal ne contient plus que : Accueil, À propos, Parcours, Projets, Compétences, Contact.
- Associations, centres d’intérêt et bénévolat sont maintenant rangés dans la section À propos.
- La section Compétences ne contient plus que les deux compétences de BUT1 GEII :
  - C1 — Concevoir la partie GEII d’un système industriel
  - C2 — Vérifier la partie GEII d’un système industriel
- Les barres de progression ont été supprimées.
- Chaque compétence contient des menus déroulants avec les apprentissages critiques AC11.01, AC11.02, AC11.03, AC12.01, AC12.02 et AC12.03.
- Chaque apprentissage critique renvoie vers la partie correspondante de la page `pages/projet-rafale.html`.
- Des boutons “Retour aux compétences” ont été ajoutés sur la page projet.

## Ajouter les images

Voir le fichier :

`assets/images/GUIDE_IMAGES.md`

Tu peux mettre toutes tes images dans :

`assets/images/photos-a-ajouter/`

Puis demander à Codex de les placer automatiquement avec ce prompt :

```text
Toutes mes images sont dans assets/images/photos-a-ajouter/. Remplace les images provisoires du portfolio par les vraies images quand le nom du fichier correspond au contenu. Garde les images provisoires quand tu n'es pas sûr. Ne change pas la structure du site. Mets aussi à jour les textes alt des images.
```

## Remplacer le CV

Remplacer le fichier :

`assets/cv/cv-diane-lanau.pdf`

par le vrai CV, en gardant le même nom de fichier.
